This directory contains shell scripts that demonstrate basic Linux commands.

0-current_working_directory prints the absolute path of the current working directory using the pwd command.

1-listit displays the contents of the current directory using ls.

2-bring_me_home changes the working directory to the user's home directory using the cd command.

3-listfiles displays the contents of the current directory in long format using ls -l.

4-listmorefiles displays all files (including hidden ones) in long format using ls -la.
